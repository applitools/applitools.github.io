Open Source @ Applitools

Appium

Applitools did significant work and continues to help with the Appium project in complex features such as deterministic scrolling and many others.

Cypress SDK

The Cypress SDK is a plugin for Cypress.io OSS project. The SDK takes DOM snapshots and sends them up to Applitools Visual Grid for Visual testing and monitoring purposes.

Selenium IDE

Applitools is the main sponsor and contributor to Selenium IDE. Applitools, together with the rest of the Selenium community and the SideeX project, have been working closely to release this new version.

Storybook SDK

The Storybook SDK is a test runner for React, Vue, and Angular projects that use Storybook for visual documentation. The SDK opens all the components by opening Storybook and takes DOM snapshots and sends them up to Applitools Visual Grid for Visual testing and monitoring purposes.
